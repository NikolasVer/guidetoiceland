## Technology Stack
 #### Server:
   - NodeJS + Express
   - GraphQL + Apollo Server
 #### Client
   - React
   - Apollo client
   - Jest + Enzyme

## Setup 
- npm i
- cd client/ && npm i

## Run dev
- npm run server:dev
- cd client/ && npm start
